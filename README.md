# Infrastructure PrestaShop Docker - Production Ready

## Installation

1. Cloner le dépôt :

2. Copier et configurer le fichier d'environnement :

```bash
cp .env.example .env
```

3. Lancer l'infrastructure :

```bash
docker-compose up -d
```

4. Vérifier que tous les services fonctionnent correctement :

```bash
docker-compose ps
```
